# Node File transfer script
A basic file transfer script made in Node to share files using http

## How to install
- Step 1: Clone this repository
- Step 2: Configure your desired folder and PORT in `index.js`
- Step 3: run `node index.js`
- Step 4: Navigate to `localhost:{PORT}`

## Notice!
This script is unsafe for public networks, i only recommend using it on trusted networks
